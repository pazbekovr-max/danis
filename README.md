# Expense Tracker (Трекер расходов)

## Описание
Простое и эффективное GUI-приложение для отслеживания личных расходов, разработанное с использованием Python и библиотеки Tkinter. Приложение позволяет вносить записи о тратах, фильтровать их по категориям и датам, а также автоматически подсчитывать итоговую сумму за выбранный период.

## Основные функции
1. **Ввод данных:** Добавление расходов с указанием суммы, категории и даты.
2. **Валидация:** Проверка на положительную сумму и корректный формат даты (ДД.ММ.ГГГГ).
3. **Фильтрация:** Просмотр расходов по конкретной категории или в диапазоне дат.
4. **Статистика:** Мгновенный подсчет общей суммы отфильтрованных записей.
5. **Хранение данных:** Сохранение и загрузка данных в формате JSON.
6. **Контроль версий:** Интеграция с Git для отслеживания изменений кода.
